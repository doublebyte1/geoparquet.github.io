GeoParquet is rapidly maturing, with a number of new software libraries and tools coming online.

Libraries

  • GeoPandas (Python)
  • geoarrow (R)

  • GDAL/OGR (C++, bindings in several languages)
  • GeoParquet.jl (Julia)
