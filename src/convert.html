GeoParquet <-> GeoJSON

Convert GeoJSON to GeoParquet and vice versa without leaving your browser.

This page uses the gpq utility to read and write GeoParquet files. The conversion is done using a WebAssembly build of gpq. Since the conversion is done in the browser, you may run into issues with very large files. To work with larger files, download the gpq program to do the conversion.

See the section on known limitations for details on what to expect. Please open an issue if you run into problems.
